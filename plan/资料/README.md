## 使用规则

1. 本目录文档仅作背景参考，不替代 `plan/规格` 与任务目录中的正式方案文档。
2. 根目录发现同名资料时，保留本目录版本，删除根目录重复副本。
3. 文档更新后请在对应任务目录的 `任务清单.md` 记录变更原因，避免后续语义漂移。
