# 下线聊天页并精简更新页

创建时间：2026-02-26 23:55:00

## 为什么做

龙虾 Bot 已自带 Web UI 对话页，控制面板继续保留“智能对话”页面会造成重复入口；更新页同时存在两套更新能力也会增加误操作。

## 改什么

- 删除 `chat-console` 页面路由、页面文件以及侧栏“智能对话”入口。
- 删除 `/update` 页面中的“龙虾 Bot 直装更新”整块，仅保留“龙虾控制台版本更新”。
- 统一更新页两处核心提示文案：强调“公开仓库 release -> 先准备版本包 -> 再应用更新并重启”。

## 影响范围

- 涉及模块：`public/pages/*.html`、`public/app.js`、`public/app-routes.js`、`src/server.js`、`public/js/pages/system-page.js`
- 涉及接口：前端不再触发 `target=bot` 的更新页面动作；后端 API 保持兼容
- 风险等级：L1

## 不做什么（边界）

- 本次不删除后端 Bot 直装更新 API（保留兼容能力，避免影响其他调用方）。
- 本次不调整 Gateway、模型、渠道、服务控制等其他页面功能。
- 本次不做样式大改，仅做结构收敛与文案统一。
