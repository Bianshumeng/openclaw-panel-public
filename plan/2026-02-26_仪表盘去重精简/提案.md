# 仪表盘去重精简

创建时间：2026-02-26 23:59:00

## 为什么做

仪表盘存在与侧栏和其他页面重复的入口与信息，用户反馈页面冗余且干扰主流程，需要收敛为必要信息和关键动作。

## 改什么

- 删除顶部统计中的“渠道 / 技能 / 错误 / 当前版本”。
- 删除“我想要”中的重复快捷按钮（配置模型、配置平台接入、管理技能、检查版本更新、服务控制）。
- 删除“地址与接入配置”和“运行态明细”整块。
- 保留并强调 Gateway 运维动作与模型快速切换。

## 影响范围

- 涉及模块：`public/pages/dashboard.html`、`public/index.html`、`plan/规格/OpenClaw面板.md`
- 风险等级：L1

## 不做什么（边界）

- 不修改网关 Token 自动审批逻辑。
- 不修改模型配置、渠道配置、技能管理、版本更新、服务控制页面本身功能。
- 不做接口变更，仅做仪表盘信息收敛。
