模型配置

选择你需要的操作：切换已有模型，或添加新的模型服务。

切换默认模型

选择要使用的默认模型，然后点击保存。

提示：这里只切换默认模型，不会新增或删除模型服务。

可用模型

在这里直接按提供商查看并切换模型。切换前建议填写当前会话上下文，系统会在可能超限时提醒你。

当目标模型上下文更短，且当前会话上下文超过目标上限时，将弹出风险提示。

添加新模型服务

基础配置适合绝大多数用户；只有你需要自定义 models JSON 时，再切换到自定义模式。

当前为快速配置模式：你通常只需要填写服务名称、接口地址和密钥。

快速配置

选择模型类型，填写基本信息即可使用。

高级参数（通常不用改）

提示：保存后默认只新增模型服务；勾选「保存后设为当前默认模型」才会同时切换默认模型。

高级模式

需要完全自定义时，直接编辑模型 JSON 数组。每个模型至少要有 id 和 name。
