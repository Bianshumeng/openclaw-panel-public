## 记录规则

1. 每次迁移动作单独建一份文档，命名建议：`YYYY-MM-DD_机器标识_迁移记录.md`。
2. 文档至少包含：目标机器、执行人、执行时间、变更内容、验证结果、风险与回滚点。
3. 发现可复发问题时，必须写“问题现象 -> 根因 -> 处理 -> 防复发规则”。
